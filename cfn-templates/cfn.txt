Create a CloudFormation template with embedded templates that automates the provisioning & configuration of an environment (dev, qa, test, staging & prod).

aip-bi-master.cfn.json - The master template that consist of sub-templates described below.

aip-bi-vpc.cfn.json - This template defines the VPC for the environment

aip-bi-admin-vpc.cfn.json - This template defines the Admin VPC & VPN and configure VPC peering with env VPC

aip-bi-admin-resources - This template defines the AWS resources for the Admin VPC (JFrog, Docker Trusted Repo, Jenkins)

aip-bi-rds.cfn.json - This template defines the RDS setup

aip-bi-cache.cfn.json - This template defines the ElastiCache setup

aip-bi-aws-resources.cfn.json - This template defines all the AWS dependencies for the Elastic Beanstalk apps (CloudWatch groups, S3 bucket, Lambda function, SQS queues)

aip-bi-eb-bi-app.cfn.json - This template defines the Beanstalk application (web tier) to deploy the BI app (lb, web, api & data services)

aip-bi-eb-etl-app.cfn.json - This template defines the Beanstalk app to deploy the ETL app (worker tier)
